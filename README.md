# oh-my-rime-wanxiang-linux
以薄荷输入法为底座,万象输入法为扩展词库的RIME输入法配置
本配置建议与薄荷输入法自动配置脚本文件搭配使用,这是自动配置Fcitx5-RIME脚本工程中的一部分
使用的万象词库:https://github.com/amzxyz/rime_wanxiang/releases
使用的万象语言模型:https://github.com/amzxyz/RIME-LMDG/releases
